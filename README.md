# สบายโฮม

Thai storefront landing page for a direct-to-consumer ecommerce brand.

## Run Locally

```bash
python3 -m http.server 4173
```

Open `http://127.0.0.1:4173`.

## Admin

Open `http://127.0.0.1:4173/admin.html`.

Demo password:

```text
admin123
```

## Deploy

The project includes a `Dockerfile` for Railway. Railway should build the image and run the static site on its injected `PORT`.
